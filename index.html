About the System

🛵

What is Sakay Partido?

Sakay Partido is a transportation mobile application designed to help students and commuters easily find available rides and routes going to Partido State University. It connects riders with motorcycle (door-to-door) and tricycle services, making daily commuting faster, safer, and more convenient.

🎯

Goal

Evaluate the usability and user experience of the Sakay Partido prototype via System Usability Scale (SUS).

👥

Respondents

Students from Partido State University, ages 18–24, experienced in using smartphones and commuting daily.

📊

Method

Heuristic evaluation by group members + SUS questionnaire answered by user participants.
